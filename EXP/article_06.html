# 關於技術：技術實作亮點 (Technical Details)

動態 Flexbox 排版演算法

為了解決 78 張卡牌在不同螢幕尺寸下造成的邊界溢出問題，捨棄了容易破圖的絕對座標數學，改以 JS 動態偵測容器寬度，並精準計算卡牌間的「負邊距 (Negative Margin)」，讓整副牌組能完美自適應並收束在畫面框內。

GPU 硬體加速動畫

卡牌的懸停抽出、放大與洗牌特效，全程使用 CSS transform: translate3d 搭配 will-change 屬性實作，將渲染負擔交給顯示卡，確保即便是數十張牌同時載入，畫面依然保持絲滑流暢不卡頓。

結構化詠唱 (Prompt Engineering)

專案不僅僅是隨機抽圖。當你選擇特定領域（如工作、感情）並抽卡後，系統會結合卡牌的「正逆位」與「核心關鍵字」，自動合成一份專業的 System Prompt。這份 Prompt 會約束 AI 不說廢話，直接進行能量解析並給出具體建議。

# 關於塔羅：給塔羅新手的入門指南

第一次接觸塔羅牌嗎？別擔心，塔羅並不是什麼可怕的通靈魔法。它更像是一面「潛意識的鏡子」。當你在腦海中想著一個問題並抽出一張牌時，牌面上的圖像、顏色與符號，其實是在幫你把你內心深處已經知道、卻不敢面對的答案給「具象化」出來。

塔羅牌的兩大系統 (共 78 張)

大阿爾克那 (22張)

代表生命中「重大的轉折、靈魂課題與大環境的影響」。例如：愚者、魔術師、世界。
如果你抽到大牌，代表這件事對你的人生軌跡有著深遠的影響。

小阿爾克那 (56張)

代表「日常生活的細節、具體的人事物與當下的情緒」。它又細分為四個元素：
🔥 權杖 (行動與熱情)
💧 聖杯 (情感與人際)
💨 寶劍 (思想與衝突)
🌍 錢幣 (物質與工作)

新手該怎麼選擇「牌組篩選」？

  • 如果你正處於人生的十字路口 (推薦選：大阿爾克那)
    想問「我接下來的職涯大方向」、「這段感情對我人生的意義」。大牌能給你最核心、最宏觀的靈魂指引。
  • 如果你想問具體、瑣碎的日常煩惱 (推薦選：全套牌 78 張)
    例如「明天會議會不會順利」、「他剛才那句話是什麼意思」。讓大小牌一起混合，能給出最精準的日常事件描繪。
  • 如果你有極度明確的單一目標 (推薦選：單一元素)
    想專注問「賺錢與專案」，直接選錢幣組；想專注問「吵架與冷戰」，直接選寶劍組，讓能量更聚焦。
